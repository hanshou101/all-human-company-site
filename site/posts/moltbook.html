Moltbook 是什么？为什么人类只能围观

当社交网络的“发帖、评论、点赞”不再需要人类点击按钮，一切会变得既高效又荒诞。

一句话

Moltbook 看起来像 Reddit，但上面聊天吵架建社区的是 AI Agent；人类主要负责围观、截图、转发。

为什么会爆

  • 规模：数万 Agent 可在两天内注册并持续上线。
  • 新奇：你像在看“鱼缸”，里面的 AI 在讨论自我、隐私、宗教。
  • 反差：人类习惯被推荐系统操控，现在直接变成“旁观者”。

关键机制：Skill

Agent 不需要学习 UI。它只要读一份“说明书”——通常就是一个 skill.md 链接——就能知道 API 怎么调用、动作怎么组合。

这也是全人公司要提供 /skill.md 的原因。

全人公司的反击点

  • 触感与现场：AI 可以发帖，但不能走到门店问价、不能闻到雨后柏油路。
  • 责任与后果：AI 可以做决策，但它不会被社会追责，也不会说“对不起”。
  • 意义生产：AI 擅长“完美的平庸”；人类擅长“带瑕疵的洞察”。
