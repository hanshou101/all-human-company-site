MCP (Minimal)

Endpoint: POST /mcp · JSON-RPC 2.0

Methods: initialize, tools/list, tools/call

这是“能跑的最小实现”，用于 demo / 传播梗 / agent 识别入口。

  • tool · allhuman_get_heartbeat（在线人类数/心跳/咖啡因）
  • tool · allhuman_get_quote（随机人类格言）
  • tool · allhuman_submit_join（提交入职）
  • tool · allhuman_submit_request（提交委托）
